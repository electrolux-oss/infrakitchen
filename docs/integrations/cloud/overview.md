# Cloud Providers

This section provides an overview of how the platform integrates with leading cloud providers.
Each integration allows you to connect, manage, and automate workloads across different cloud environments using a consistent interface and unified tooling.

### Supported Cloud Providers

The following cloud providers are currently supported:

- **[AWS (Amazon Web Services)](aws.md)**
  Connect and manage AWS resources including EC2, S3, Lambda, and more.

- **[Azure (Microsoft Azure)](azure.md)**
  Seamlessly integrate with Azure services such as Virtual Machines, Storage Accounts, and Azure Functions.

- **[Google Cloud (GCP)](gcp.md)**
  Manage Google Cloud resources like Compute Engine, Cloud Storage, and Cloud Functions.

- **[MongoDB](mongodb.md)**
  Integrate with MongoDB Atlas for automation and management of your database clusters.

- **[Datadog](datadog.md)**
  Connect to Datadog for configuration to monitor your cloud infrastructure and applications.
